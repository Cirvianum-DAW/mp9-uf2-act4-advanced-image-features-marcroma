Palmeres

1. Convert the image to SVG format.


2. Isolate one of the palm trees.

Step 1 - Choose the Selection Tool.

Step 2 - Use the Local Element Selector.

Step 3 - Draw a square and select the palm tree's elements.



3. Change the colors of the palm tree leaves, creating a different color range.

Firstly, through the application.

Step 1 - Select the leaf we want to modify.

Step 2 - Select the new color from the picker.


Secondly, through the text editor.

Step 1 - Open the text editor for the image.

Step 2 - Choose the element.

Step 3+4 - Change the style # Color Code.
